# VC++GoogleTestDemo

Visual Studio で C++ テストの仕組みを作ってみたかった。



**できたこと**

- 静的ライブラリ (*.lib) プロジェクトのテスト



**できていないこと**

- 動的ライブラリ (*.dll) プロジェクトのテスト
